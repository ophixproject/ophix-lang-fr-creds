# ophix-lang-fr-creds

French translation pack for [ophix-creds](https://ophix.io).

## Install

```
pip install ophix-lang-fr-creds
```

Then set `LANGUAGE_CODE = "fr"` (or `"fr-fr"`) in your server `.env`.

## What is translated

- Admin UI strings: app name, inline labels, column headers, fieldset titles
- Help text strings from `ophix-server-base` (ClientArtifactBase fields)
